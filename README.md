# SciMLGA 2026 Short Course Materials

This repository contains the notebooks, scripts, and supporting materials used in the short course **"Deep Learning for Real-World Geophysical Problems: Challenges in Seismic Methods"**, taught by **Dr. Ana Muller (Petrobras)** during

> **SciMLGA 2026 – Scientific Machine Learning for Geophysics and Astronomy**  
> Observatório Nacional (ON/MCTI)  
> Rio de Janeiro, Brazil  
> **June 15–19, 2026**

All scripts, notebooks, and accompanying materials were developed by **Dr. Ana Muller** and are made publicly available to support learning and reproducible research.
